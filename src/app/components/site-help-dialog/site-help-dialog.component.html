Icon Reference

▶ Launch — start a new run of this checklist
▶▶ Continue — resume an in-progress run
✎ Edit — edit the checklist items, name, or description
✕ Delete — permanently remove this checklist
▶ Expand — show or hide checklist details

The List Name is required and should clearly describe the process or task you want to repeat, such as:

  • RV Departure Checklist
  • Monthly Server Maintenance
  • Pre-flight Checklist
  • Camping Setup

The Description field is optional and can be used to add additional context or notes about when the checklist should be used.

After clicking Create, you will be taken to the next page where you can begin adding checklist items to the list.

Use the Create New List button to create a new checklist.

For each existing list you may:

  • Click the pencil icon to edit the checklist items, name, or description
  • Click the green launch button to begin a new checklist run
  • Click the X button to permanently delete the checklist

If a checklist already has a run currently in progress, the launch button will appear yellow with two arrows instead of the normal green launch icon.

Deleting a checklist is permanent. You will be asked to confirm deletion before it is removed. Deleting a checklist will also permanently remove:

  • Checklist items
  • Current runs
  • Run history
  • Completed item history

Deleted checklists cannot be recovered.

Repeatable checklists are designed to be reused over and over again. Launching a checklist creates a new checklist run while keeping the original checklist intact for future use.

This page is where you build and maintain the items for your repeatable checklist.

You may add as many checklist items as needed. Each item represents a task or step that should be completed when running the checklist.

Examples:

  • Disconnect power cable
  • Lock storage compartments
  • Verify backup completed
  • Check tire pressure

Items can be reordered using the drag handles on the right side of each row.

To edit the checklist name or description, click the pencil icon next to the list title.

The checklist itself is permanent and designed to be reused as many times as needed. Unlike a one-time grocery list, completing a checklist run does not remove or delete the checklist.

When your checklist is ready, click the Launch button to begin your first checklist run.

You may also:

  • Return to the main lists page using the Back to Lists link
  • Edit the checklist again later at any time
  • Re-run the checklist as often as needed from this page or the main lists page
