Icon Reference

▶ Launch Launch — start a new run
▶▶ Continue Continue — resume an in-progress run
Edit — open list details and items
Delete — permanently remove
▶ Expand — show or hide details
Drag handle — reorder items

Your lists page is the home screen of zChecklist. Use the Create New List button to add a new checklist.

For each list you can:

  • Click the expand arrow (▶) to show run history, item count, and description
  • Click the edit icon to manage items, members, name, or description
  • Click the launch icon to start a new run
  • Click the continue icon to resume a run that is already in progress
  • Click the trash icon to permanently delete the list

An In Progress badge appears on any list that has an active run. Only one run can be active at a time per list.

When expanded, a History link appears for lists that have completed runs.

If another user has shared a list with you, it will also appear here alongside your own lists.

Deleting a list is permanent and will also remove all items, runs, and history. You will be asked to confirm before deletion.

The List Name is required and should clearly describe the process you want to repeat, such as:

  • RV Departure Checklist
  • Monthly Server Maintenance
  • Pre-flight Checklist
  • Camping Setup

The Description is optional. Use it to add context about when or how the checklist should be used.

After clicking Create, you'll be taken to the list detail page where you can start adding items.

The edit page has two tabs: Items and Members.

Items tab:

  • Add as many items as needed — each represents one step or task
  • Use the drag handle on the right of each row to reorder items
  • Check One-time on an item if it should only appear once and not repeat in future runs
  • Click the trash icon on an item to remove it

Click the pencil icon next to the list title to edit the name or description.

Members tab (Premium):

  • Invite other users to collaborate on this list by email
  • Pending invitations are shown until accepted or expired
  • Remove a member at any time
  • The list owner's first included collaborator is free; additional collaborators are $1/month each

When your list is ready, click Launch to start your first run.

Tap any item to check it off. Tap it again to uncheck it.

Use Check All at the top to mark every item complete at once.

Items marked as one-time will be removed from the list after this run — they will not appear in future runs.

On shared lists, a small initials badge appears next to each completed item showing who checked it off.

When all items are done, tap Complete List to finish the run. The completion is saved to your run history and the list resets automatically for next time.

You can leave a run in progress and come back to it later — your progress is saved. Use the Continue button from the Lists page to resume.
