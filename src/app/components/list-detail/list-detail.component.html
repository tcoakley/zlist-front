This page is where you build and maintain the items for your repeatable checklist.

You may add as many checklist items as needed. Each item represents a task or step that should be completed when running the checklist.

Examples:

  • Disconnect power cable
  • Lock storage compartments
  • Verify backup completed
  • Check tire pressure

Items can be reordered using the drag handles on the right side of each row.

To edit the checklist name or description, click the pencil icon next to the list title.

The checklist itself is permanent and designed to be reused as many times as needed. Unlike a one-time grocery list, completing a checklist run does not remove or delete the checklist.

When your checklist is ready, click the Launch button to begin your first checklist run.

You may also:

  • Return to the main lists page using the Back to Lists link
  • Edit the checklist again later at any time
  • Re-run the checklist as often as needed from this page or the main lists page
